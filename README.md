
# Expense Tracker (React)

A simple expense tracker built with React that allows users to manage income and expenses with local storage support.


##  Features
- Add income and expense transactions
- Delete transactions
- View balance, income, and expense summary
- Data persists using local storage

##  Tech Stack
- React
- Vite
- JavaScript
- CSS
- Local Storage

##  Project Structure
src/
├─ components/
├─ App.jsx
├─ App.css
├─ main.jsx

## Run Locally
git clone https://github.com/aminul441/expense-tracker-react.git
cd expense-tracker-react
npm install
npm run dev
